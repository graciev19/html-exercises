Step 1: Understand Basic HTML Coding Elements

Main Content Categories

Categories that consist of common rules shared by various elements used to establish of "flow" of HTML elements. This includes...

  • the "base" command
  • the "link" command
  • the "title" command

Specific Content Categories

Categories that consist of rare categories shared only be a few elements used to establish of "flow" of HTML elements. This includes...

  • the "mark" command

  • the "meta" command
